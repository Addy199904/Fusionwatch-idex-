# Fusionwatch-idex-
AI-Native Satellite Intelligence System — iDEX Challenge #4
## ⚠️ Publication Notice

This repository contains FusionWatch v0.1 — a proof-of-concept 
prototype trained on publicly available DOTA v1.5 dataset 
(926 images, civilian aerial imagery).

**Why a limited release:**
The full FusionWatch system integrates Indian satellite data 
(Cartosat-3, RISAT-2B), SAR-optical fusion, and military-specific 
training data. For operational security reasons, only the 
baseline detection prototype is published here.

This release demonstrates:
- Technical feasibility (0.502 mAP50 on satellite imagery)
- System architecture and pipeline design
- Real-time inference capability (13.8ms/image)

The complete system is being developed under iDEX Challenge #4 
with full capabilities contingent on data access and 
computational infrastructure funding.

**Submitted to:** Innovations for Defence Excellence (iDEX), 
Ministry of Defence, Government of India
